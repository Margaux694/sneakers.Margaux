# Site web de chaussures – Candidature Web@cadémie EPITECH

## Présentation
Ce site web est un **projet personnel réalisé en autonomie** dans le cadre de ma formation actuelle en développement web.  
Il a été conçu **de A à Z** et est présenté dans le cadre de ma **candidature à la Web@cadémie d’EPITECH**, afin de démontrer ma motivation et mes compétences techniques.

Le site a pour thème la **présentation et la mise en valeur de chaussures**, avec un affichage dynamique des contenus.

## Objectif du projet
Les objectifs de ce projet étaient de :
- créer un site web complet sans suivre un tutoriel imposé
- mettre en pratique les bases du **HTML, CSS et JavaScript**
- utiliser des **données externes via un fichier JSON**
- structurer un projet web clair et fonctionnel

## Technologies utilisées
- HTML5  
- CSS3  
- JavaScript  
- JSON (données utilisées pour l’affichage du contenu)

## Fonctionnalités présentes
Le site respecte les éléments demandés pour la candidature :
- ✅ Header avec un menu de navigation
- ✅ Insertion d’images de chaussures
- ✅ Plusieurs pages reliées entre elles par des liens
- ✅ Contenu dynamique généré à partir d’un fichier JSON

## Ce que j’ai appris grâce à ce projet
Ce projet m’a permis de :
- mieux comprendre la structure d’un site web
- manipuler le DOM en JavaScript
- exploiter un fichier JSON pour afficher des données
- organiser correctement les fichiers d’un projet
- relier plusieurs pages entre elles

## Améliorations possibles
Avec plus de temps, je pourrais :
- améliorer le design et l’ergonomie
- ajouter un système de filtres ou de recherche
- rendre le site encore plus interactif
- optimiser l’affichage pour mobile
